# ReVitra Studio Bot

Telegram-бот для записи клиентов в студию восстановления автомобильных стекол ReVitra Studio.

## Возможности

- запись на услугу через пошаговый диалог;
- выбор даты (сегодня, завтра, послезавтра или произвольная дата);
- сохранение заявок в SQLite;
- проверка занятости даты (одна активная запись на день);
- просмотр и отмена своих записей;
- уведомление администратора о новой заявке.

## Требования

- Python 3.11+
- Telegram Bot Token от [@BotFather](https://t.me/BotFather)
- Telegram ID администратора

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Скопируйте `.env.example` в `.env` и заполните переменные:

```env
BOT_TOKEN=123456:ABC...
ADMIN_ID=987654321
```

`ADMIN_ID` — ваш числовой Telegram ID. Его можно узнать у [@userinfobot](https://t.me/userinfobot).

## Запуск

```bash
python main.py
```

При первом запуске создаётся файл базы данных `revitra.db`.

## Структура проекта

```
app/
  handlers/      # обработчики команд и сообщений
  keyboards/     # клавиатуры
  data/          # каталог услуг
  services/      # бизнес-логика записи
  storage/       # временное состояние диалога
  utils/         # работа с датами
  database.py    # SQLite
  config.py      # настройки из .env
main.py          # точка входа
```

## Примечания

- незавершённые заявки хранятся в памяти и сбрасываются при перезапуске бота;
- подтверждённые записи сохраняются в SQLite;
- отмена записи помечает её как `cancelled`, не удаляя из базы.
